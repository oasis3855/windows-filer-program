## 機能の概要

***このソフトウエアは個人的にプログラム開発時に利用したものです***

ファイルの一部分をコピーするためのツール。コピー開始・終了のファイルでの位置をファイル先頭からのバイト単位で指定できます。エラーが発生した場合に終了するか、読み飛ばすかの指定も出来ます。ソフトウエア作成時に、テスト用のファイルを作成する目的で作成されています。
