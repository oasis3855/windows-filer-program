【はじめに】
このプログラムは、キーワードでファイルを切り分けるツールです
当初の目的は、壊れたメモリーカードより画像データを復活するためのツール類のひ
とつです


【動作環境】
Windows 95/98/Me/NT/200/XP (日本語版および各国語版)
MFC42.DLL および MSVCRT.DLL がシステムフォルダに存在すること（Visual C++ 6.
0 対応版） 古いバージョンのDLLの場合、起動できません。


【インストール】
この画面を見ているということは、すでにインストール完了です


【アンインストール】
このプログラムは、レジストリおよび設定ファイルなどを利用しません
プログラム本体を消去すればそれで終わりです


【利用法】
元ファイル ： 切り出し元のデータファイルを指定します
切り出し先フォルダ ： 切り出したデータを出力するフォルダを指定します
  （フォルダ内には他のファイルが存在しないのが好ましい）
キーワード ： 16進数8バイトのバイト列を記入します
  例)EXIFファイルを切り出すときは 「FF D8 FF E1」を指定します
  16進数を直接記入しても、0x00のように C言語スタイルでも、00Hのようにアセン
  ブラスタイルでも記入できます。文字の間はスペースで区切ってください
必要な設定が終われば、「実行」ボタンを押します。640MBytes程度のファイルなら、
約2〜3分程度で処理が終了します

rescue-data.html に利用方法の概略が記述されています

ユーザはある程度コンピュータに関する知識があることが要求されます
